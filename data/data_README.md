# Data

このプロジェクトでは、元データとして UCI Individual Household Electric Power Consumption Dataset を使用しています。

元データはサイズが大きいため、このリポジトリには含めていません。

## データの入手方法

以下の公式ページからダウンロードしてください。

- UCI Machine Learning Repository
- Individual Household Electric Power Consumption Dataset

## 配置方法

ダウンロード後、以下のように配置してください。

```text
data/
  raw/
    household_power_consumption.txt
